# BASIC CHAT GUI WITH PYTHON

## Installation
```
git clone https://github.com/sodtiruk/basic_gui_chat_online.git
cd basic_gui_chat_online
```
## Usage
### Open Server
```
python server.py
```
### Join Server
If you want to send a message to yourself, you must join server 2 times.
```
python client.py
```
